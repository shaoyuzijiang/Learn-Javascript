    Mac软件收集

    # Mac软件收集

    # 生产力、效率工具

    • Brew：Mac开发必备，命令行下载软件
    • Brew Cask：搭配Brew使用
    • Alfred：Mac开发必备，提高效率神器
    • iTerm2：Mac开发必备
    • Dash：文档查询
    • oh-my-zsh：zsh一出，无人再与争锋!如果你还在用bash，请立马放弃，投向zsh的阵营
    • Cheatsheet：快键键
    • Gas Mask (opens new window): 快速编辑和切换hosts文件
    • Resume Time：记录时间都去哪儿
    • Haze Over: 背景窗口虚化，突出当前应用的焦点，避免打开应用太多，找不到当前应用
    • Just Focus：25分钟的番茄工作法，结束强制锁屏休息
    • PDF Expert 2：阅读PDF的强力助手
    • Spectacle (opens new window)：窗口管理软件（免费）
    • Magnet (opens new window): 窗口管理，拖动到屏幕边缘即可自动实现全屏、1/2屏、1/3屏
    • Sip (opens new window): Color Picker for developers
    • SpaceLauncher (opens new window): 以 macOS 空格键为核心创建你的热键阵营 (opens new window)
    • Shuttle (opens new window)：免费的简易的SSH快捷菜单
    • Sip：取色器
    • https://www.nuoshell.com/: nuoshell, Mac ssh 客户端工具

    # 笔记软件

    • Typora (opens new window): 一款简洁的 markdown editor。简洁的 UI，良好的写作体验。

    # 实用工具

    • xnip (opens new window): Mac 上方便好用免费的截图工具。支持截图标注、截图固定、滚动截图。还支持截图阴影，这点太赞了。
    • AppCleaner：彻底删除 Mac App
    • Typetto (opens new window)：键盘控制。通过蓝牙连接，Mac的键盘控制iPad、iPhone的输入
    • Resize Master (opens new window)：免费压缩图片
    • vanilla 官网下载 (opens new window): 精简 Mac 菜单栏应用图标，小巧轻量还免费，评测见 Vanilla，精简 Mac 菜单栏应用图标，小巧轻量还免费丨App+1 (opens new window)
    • AirServer (opens new window)：将 iPhone 屏幕通过 AirPlay 投射到 Mac 的屏幕上
    • Mos (opens new window)：一个用于在 MacOS 上平滑你的鼠标滚动效果或单独设置滚动方向的小工具, 让你的滚轮爽如触控板
    • Better Scroll (opens new window): 鼠标滚动
    • Lunar (opens new window): 轻松调节外接显示器的亮度
    • PP鸭 (opens new window): 图片压缩神器
    • omnidisksweeper (opens new window): 磁盘文件扫描，发现Mac系统占用空间大的文件夹
    • PicGo (opens new window): 图片上传管理软件，可对接国内几个主流的图床。
    • The Unarchiver (opens new window) 免费好用的文件解压工具
    • SwitchKey (opens new window): 可以根据当前 App 窗口自动切换指定的输入法。当需要在代码与中文输入软件之间切换时，特别好用。
    • Kap (opens new window): Mac 免费轻量 GIF 录屏小工具。
    • Clean Me (opens new window): A macOS system analyzer and cleaner。
    • MonitorControl (opens new window)：控制外接显示器的屏幕亮度和声音。

    # 写作工具

    • Ulyless：强大的写作工具
    • MacDown：Markdown写作必备，方便转换成PDF等格式
    • 1Checker (opens new window): 英语拼写检查，避免尴尬错误

    # Something Cool

    • Fliqlo：翻盖时钟屏保，Mac的经典屏保
    • Brooklyn (opens new window): Mac 酷炫屏保。Screensaver inspired by Apple's Event on October 30, 2018
